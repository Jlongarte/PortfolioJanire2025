## Portfolio Personal 

Bienvenido a mi portfolio personal, un proyecto desarrollado con Vite, HTML, CSS y JavaScript (DOM).
He elegido una temática del espacio para reflejar mi interés por el diseño visual, las animaciones y la experiencia de usuario.

 ## Descripción

Este portfolio funciona como una presentación interactiva de mis habilidades, proyectos y estilo de desarrollo.
Incluye diversas animaciones y efectos visuales inspirados en el universo, con una navegación fluida y dinámica.

Cada sección está diseñada con detalle, buscando transmitir una sensación de exploración y movimiento. El objetivo principal es combinar creatividad y funcionalidad en una experiencia moderna y atractiva.

## Tecnologías utilizadas

Vite — entorno de desarrollo rápido y moderno

HTML5 — estructura del contenido

CSS3 — estilos y animaciones

JavaScript (DOM) — interactividad y manipulación dinámica

Animaciones con CSS y JS — efectos de movimiento, transiciones y fondo espacial

## Proyectos incluidos

El portfolio presenta cuatro proyectos destacados, cada uno con su propia sección y diseño individual:

- Proyecto ChillFlix 

- Proyecto X clone

- Proyecto Spotifly

- Proyecto Página Web CrazyBurguer


## Características principales

Diseño responsive, adaptable a diferentes dispositivos y pantallas

Animaciones y transiciones fluidas

Navegación intuitiva con ambientación espacial

Código limpio, modular y comentado

Optimización de rendimiento con Vite
